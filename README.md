# Enterprise DevOps CI/CD Platform

## Overview

This repository demonstrates an end-to-end DevOps pipeline using
Jenkins, SonarQube, Docker, Prometheus, and Grafana.

## Key Features

- Pipeline-as-Code using Jenkinsfile
- Automated code quality checks
- Docker-based immutable artifacts
- Monitoring & observability

## Trigger Flow

GitHub Push → Webhook → Jenkins → SonarQube → Docker → Monitoring

## Audience

Senior management, clients, DevOps engineers
